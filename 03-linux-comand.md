БАЗА:

cd - перейти в директорию
mkdir - создать директорию
rm/rm -rf - удалить файл/директорию
nano/vim - редактирование файла через терминал
touch - Создание файла
chmod - Изменнение прав

НАВИГАЦИЯ

ls -la - все файлы в директории
pwd - текущая директория
tree - дерево папок
which - путь к файлу (which docker)
whereis - это как which, только круче,но пока хз как именно

ПРОСМОТР

cat - весь файл в консоль
less - листать файл
head / tail - начало/конец файла (tail -f logfile)
grep - искать по файлам (grep error log.txt) 
top/htop - процессы и нагрузка на CPU/RAM
ps aux - список всех процессов
df -h - место на дисках 
du -sh * -размер папок в текущей директории
free -h - свободная оперативка
uptime - как давно работает сервер и нагрузка
dmesg - инфа про ядро хзваще

СЕТЬ

ip a - показать IP-адреса
ping - проверить доступность хоста
curl/wget - Загрузить/проверить URL
netstat -tulpn | ss -tulpn - Какие порты слушает сервер
nslookup / dig - проверка DNS

ПРАВА

whoami - хто я?
id - информация о пользователе
useradd - создать пользователя

ПРОЕССЫ

kill/killall - УНИЧТОЖИТЬ процесс по имени ли PID
systemctl - Управление сервисами
journalctl - Смотреть логи systemd-сервисов

Симовол | передаёт вывод из одной команды в другую. (Пайплайн типа)
